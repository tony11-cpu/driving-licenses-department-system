## Getting Started

1. Clone the repository
2. Open SQL Server and restore the provided `.bak` file from the `Database + Diagrams` folder
3. Name the restored database exactly `DVLD_02`
4. Update the connection string in `App.config` to match your SQL Server instance name
5. Open the solution in Visual Studio and run
6. On first launch the app will prompt you to set up credentials automatically
7. Use the default admin account to get started:
   - Username: `Admin_1`
   - Password: `123123123`
